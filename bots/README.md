# BenchmarkWatcher Bots

Telegram and Discord bots that provide commodity price data from [benchmarkwatcher.online](https://benchmarkwatcher.online).

## Quick Start

### 1. Install Dependencies

```bash
cd bots
pip install -r requirements.txt
```

### 2. Configure Tokens

Copy the example environment file and add your tokens:

```bash
cp .env.example .env
```

Edit `.env` with your bot tokens (see setup instructions below).

### 3. Run Bots

**Telegram:**
```bash
python telegram_bot.py
```

**Discord:**
```bash
python discord_bot.py
```

## Getting Bot Tokens

### Telegram

1. Open Telegram and message [@BotFather](https://t.me/BotFather)
2. Send `/newbot`
3. Choose a name: `BenchmarkWatcher Bot`
4. Choose a username: `benchmarkwatcher_bot` (must end in `bot`)
5. Copy the token (looks like `123456789:ABCdefGHI...`)
6. Add to `.env`: `TELEGRAM_BOT_TOKEN=your_token`

**Optional customization with BotFather:**
- `/setdescription` - Add a description
- `/setabouttext` - Add about text
- `/setuserpic` - Add profile picture

### Discord

1. Go to [Discord Developer Portal](https://discord.com/developers/applications)
2. Click **New Application** → Name it `BenchmarkWatcher Bot`
3. Go to **Bot** section → Click **Reset Token** → Copy token
4. Add to `.env`: `DISCORD_BOT_TOKEN=your_token`
5. Enable **Message Content Intent** in Bot settings
6. Go to **OAuth2 → URL Generator**:
   - Select scopes: `bot`
   - Select permissions: `Send Messages`, `Embed Links`, `Read Message History`
7. Copy the generated URL and open it to add the bot to your server

## Commands

Both bots support these commands:

| Command | Description |
|---------|-------------|
| `/price <commodity>` or `!price <commodity>` | Get price for a commodity |
| `/prices <category>` or `!prices <category>` | Get all prices in a category |
| `/top` or `!top` | Show top gainers and losers |
| `/list` or `!list` | Show available commodities |
| `/help` or `!help` | Show help message |

**Examples:**
- `/price brent` - Brent Crude Oil price
- `/price gold` - Gold price
- `/prices energy` - All energy commodities
- `!prices metals` - All metals (Discord)

## Hosting

### Option A: Run on cPanel (Namecheap)

⚠️ **Note:** Shared hosting typically doesn't support persistent processes. You may need to use a VPS or Railway instead.

If your cPanel supports it:
1. Upload the `bots/` folder
2. Set up a cron job or use the Python app feature

### Option B: Railway (Free Tier)

1. Push this repo to GitHub
2. Create account at [railway.app](https://railway.app)
3. New Project → Deploy from GitHub
4. Add environment variables (tokens)
5. Deploy

### Option C: Run Locally

Just run the Python scripts on your machine:
```bash
python telegram_bot.py
python discord_bot.py
```
