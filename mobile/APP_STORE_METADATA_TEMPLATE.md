## Description (recommended long form)
BenchmarkWatcher helps you monitor publicly available commodity benchmark prices in one place.

What you can do:
- Follow 70+ commodities across key market categories
- View historical trends and simple change metrics
- Compare selected commodities side by side
- Customize chart behavior and display preferences
- Export chart snapshots and copy values for reporting

Important notes:
- Data is benchmark/reference data from public sources
- This is not real-time exchange data
- The app is for informational and educational use only
- This app does not provide investment advice
