## Full Description (max 4000)
BenchmarkWatcher provides a clean mobile view of publicly available commodity benchmark prices.

Core capabilities:
- Track 70+ commodities across energy, metals, agriculture, and livestock
- Explore historical trends with selectable ranges
- Compare multiple commodities in a single chart view
- Save display preferences and chart settings locally
- Share chart images and copy values for reference

Designed for informational and educational use:
- Shows benchmark/reference data from public sources
- Not a trading platform and not real-time market execution data
- No investment advice, forecasts, or trading signals
