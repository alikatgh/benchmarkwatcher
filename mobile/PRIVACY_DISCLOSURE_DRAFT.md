# Privacy Disclosure Draft (For Store Forms)

This app displays commodity benchmark data fetched from your backend API and stores limited on-device UI preferences.

## Data handled by the app
- In-app settings/preferences (stored locally on device, e.g. selected chart period)
- Network requests to fetch commodity data from your API

## Data not directly collected by app features
- No account creation/login
- No precise location collection
- No contacts, photos, camera, microphone, or health data collection

## Notes before publishing
- Validate this draft against actual backend logs/telemetry.
- If backend collects IP addresses, request logs, or analytics, include those in store disclosures.
- Publish a public Privacy Policy URL and use that URL in App Store Connect and Google Play Console.
